# @noahyu/eslint-config-vue

> ESlint config for Vue 3 + TypeScript

## Usage

```bash
pnpm add @noahyu/eslint-config-vue
```

## Quick start

```js
// .eslintrc.js
module.exports = {
  root: true,
  extends: ['@noahyu/vue'],
  // ···
}
```
