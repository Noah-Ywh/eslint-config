# @noahyu/eslint-config-nest

> ESlint config for NestJS

## Usage

```bash
pnpm add @noahyu/eslint-config-nest
```

## Quick start

```js
// .eslintrc.js
module.exports = {
  root: true,
  extends: ['@noahyu/nest'],
  // ···
}
```
